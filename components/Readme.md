# Components for building aircrafts

This folder contains sub-folders, which contain 
aircraft components in different file-formats.

1. __aircraft__ contains sub-folders for aircrafts
containing STEP/IGES-files with aircraft geometries, like
wings, fuselage and rudders.
2. __airfoils__ contains airfoil descriptions in the
dat-format.
3. __brushless__ contains STEP/IGES-files with the
geometry of brushless engines.
4. __construction__ contains json-files that describe
the construction steps for a given aircraft or 
component.
5. __cpacs__ contains the CPACS-files for an aircraft.
6. __lipo__ contains STEP/IGES-files with the
geometry of lipo accumulator.
7. __servos__ contains STEP/IGES-files with the
geometry of servos.
