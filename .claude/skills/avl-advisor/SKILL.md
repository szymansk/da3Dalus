## Common Questions Cheat Sheet

**Q: What spacing should I use?**
→ Cosine (1.0) chordwise and spanwise for most cases. Use -sine (-2.0) root-to-tip for unswept wings without centerline chord breaks. (avl_doc:1036–1048)

**Q: Why is my solution garbage?**
→ Check: trailing vortex passing near downstream control point (Rule 1, avl_doc:1089). Ensure aligned surfaces share same spanwise spacing.

**Q: How does AVL handle compressibility?**
→ Prandtl-Glauert transformation. Valid up to M≈0.6 for unswept wings. For swept wings use Mperp = M·cos(sweep). (avl_doc:143–196)

**Q: What does Oswald factor mean in AVL output?**
→ Calculated from Trefftz-Plane CDi: $e = (C_L^2 + C_Y^2)/(\pi A \cdot C_{D,i})$. This is the inviscid/theoretical e_theo only. (avl_doc:1644–1649)

**Q: How to model a box wing?**
→ Use multiple SURFACE blocks with same COMPONENT number. Connect with vertical winglet surfaces. See Budziak §4.3 (lines 952–991) for complete example.
