## Wing Configuration

The +WingConfiguration+ is important for the generation of an aeroplane. In general it consists of segment. There is always one _root_-segment, followed by a number of _segment_s or a _tip_-segments.

[plantuml,"wing",svg]
.A wing consists of segments of +WingSegmentType+ _root_, _segment_ and _tip_.
----
@startebnf
WingConfiguration = root, {segment}, {tip};
@endebnf
----

A segment describes a part of a wing that is constructed by lofting the root airfoil of this segment to the tip airfoil of the segment. For segments of type _segment_ or _tip_ the root airfoil is equal to the previous segment's tip airfoil.
